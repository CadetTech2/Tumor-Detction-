# Tumor Detection using Machine Learning (Milestone 2)

## 📌 Project Overview

This project implements a baseline machine learning model for tumor detection using image classification techniques.

---

## 📊 Dataset

A brain MRI image dataset (image-based) was used.
The dataset contains two classes:

* Tumor
* No Tumor

---

## ⚙️ Data Preprocessing

* Image resizing (224 × 224)
* Conversion to tensor format
* Train/Test split (80/20)

---

## 🤖 Model

A Convolutional Neural Network (CNN) was used as a baseline model.

---

## 📈 Evaluation Metrics

* Accuracy
* Precision
* Recall
* F1-score

---

## 📊 Results

The model achieved reasonable performance on the dataset.
Further improvements can be made using deeper architectures.
